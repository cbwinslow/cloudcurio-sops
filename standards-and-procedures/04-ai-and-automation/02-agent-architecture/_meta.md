# Agent Architecture Standards

## Domain

This directory contains standards for AI agent architecture, including identity, context management, and inter-agent communication.

## Contents

- **naming-and-identity/** - Standards for naming and identifying agents
- **context-handling-policy/** - Policies for managing agent context and state
- **prompt-and-memory-standards/** - Standards for prompt engineering and agent memory management
- **logging-and-transparency/** - Logging requirements for agent actions and decisions
- **inter-agent-communication/** - Protocols for communication between agents

## Scope

Defines architectural patterns and standards for building robust, maintainable, and interoperable AI agents.
