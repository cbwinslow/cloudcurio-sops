# Machine Learning and LLM Standards

## Domain

This directory contains standards for machine learning model management, large language models, and ML operations.

## Contents

- **model-selection-guidelines/** - Guidelines for selecting appropriate ML models
- **finetuning-procedures/** - Procedures for fine-tuning models for specific use cases
- **bias-detection-and-mitigation/** - Standards for detecting and mitigating bias in ML systems
- **data-ingestion-policy/** - Policies for ingesting and preparing training data
- **evaluation-and-benchmarking/** - Standards for evaluating and benchmarking model performance

## Scope

Covers the lifecycle of ML models from selection through deployment, ensuring quality and responsible AI practices.
