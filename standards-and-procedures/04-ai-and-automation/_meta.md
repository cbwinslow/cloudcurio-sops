# AI and Automation Standards

## Domain

This directory contains standards for artificial intelligence, machine learning, autonomous agents, and automation systems.

## Contents

- **01-agent-behavior/** - Standards for AI agent behavior, decision-making, and ethics
- **02-agent-architecture/** - AI agent architecture, context management, and communication standards
- **03-ml-and-llm/** - Machine learning and large language model management standards
- **04-automation-and-ansible/** - Automation and Ansible configuration management standards
- **05-ai-infrastructure/** - Infrastructure standards for AI/ML workloads

## Scope

Covers the design, deployment, and operation of AI systems, autonomous agents, and automation frameworks.
