# Standards and Procedures

This directory contains the comprehensive standards and procedures documentation for CloudCurio operations.

## Structure

The standards and procedures are organized hierarchically into the following main categories:

- **00-global/** - Global standards applicable across all areas
- **01-infrastructure/** - Infrastructure-related standards and procedures
- **02-systems/** - System-level standards for servers, desktops, and laptops
- **03-software/** - Software development and application standards
- **04-ai-and-automation/** - AI, ML, and automation standards
- **05-governance/** - Governance, compliance, and organizational standards

Each directory contains a `_meta.md` file that describes its specific domain and scope.

## Usage

Navigate through the directory structure to find relevant standards and procedures for your area of work. Each subdirectory represents a specific domain with detailed instructions and guidelines.
