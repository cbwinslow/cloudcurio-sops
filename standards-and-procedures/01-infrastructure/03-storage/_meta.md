# Storage Standards

## Domain

This directory contains standards for storage management, including naming conventions, backup, recovery, and encryption.

## Contents

- **volume-naming-conventions/** - Standards for naming storage volumes and filesystems
- **backup-frequency-matrix/** - Backup frequency and retention requirements
- **recovery-verification-sop/** - Procedures for verifying backup recovery capabilities
- **zfs-and-btrfs-standards/** - Standards for ZFS and Btrfs filesystem configuration
- **encrypted-drives-policy/** - Encryption requirements and procedures for storage devices

## Scope

Covers all aspects of storage provisioning, backup, recovery, and data protection.
