# Monitoring Standards

## Domain

This directory contains standards for monitoring infrastructure health, performance, and security, including alerting and log management.

## Contents

- **alerting-severity-levels/** - Standards for alert severity classification and escalation
- **telemetry-standards/** - Telemetry collection and metrics standards
- **netdata-and-prometheus-guidelines/** - Configuration guidelines for monitoring tools
- **log-retention-policy/** - Log retention and archival policies
- **anomaly-detection-sop/** - Procedures for detecting and responding to anomalies

## Scope

Covers monitoring, alerting, logging, and observability practices across all infrastructure components.
