# Compute Standards

## Domain

This directory contains standards for compute resource management, including naming conventions, virtualization, resource allocation, and firmware management.

## Contents

- **host-naming-conventions/** - Standards for naming compute hosts and instances
- **virtualization-standards/** - Virtualization platform configuration and management standards
- **cpu-gpu-allocation/** - CPU and GPU resource allocation policies
- **power-and-performance-tuning/** - Power management and performance optimization standards
- **firmware-update-policy/** - Firmware and BIOS update procedures and policies

## Scope

Covers all aspects of compute resource provisioning, configuration, and management.
