# Linux Servers Standards

## Domain

This directory contains standards for configuring, securing, and managing Linux servers.

## Contents

- **ubuntu-baseline-config/** - Baseline configuration standards for Ubuntu servers
- **post-install-hardening/** - Security hardening procedures for newly installed servers
- **user-and-group-policy/** - User and group management policies
- **crontab-and-systemd-standards/** - Standards for scheduled tasks and system services
- **ssh-configuration-policy/** - SSH configuration and access control standards
- **logrotate-standards/** - Log rotation configuration standards

## Scope

Covers all aspects of Linux server configuration and management to ensure security and operational consistency.
