# Systems Standards

## Domain

This directory contains standards for operating system configuration, security hardening, and management of servers, desktops, and laptops.

## Contents

- **01-linux-servers/** - Standards for Linux server configuration, hardening, and management
- **02-desktops/** - Desktop workstation standards including setup, security, and software management
- **03-laptops/** - Laptop-specific standards including encryption, remote access, and data handling

## Scope

Covers operating system level standards and procedures for all system types in the CloudCurio environment.
