# Desktop Standards

## Domain

This directory contains standards for desktop workstation setup, security, software management, and maintenance.

## Contents

- **workstation-setup-sop/** - Standard operating procedures for setting up new workstations
- **security-baseline/** - Security baseline configuration for desktop systems
- **software-installation-standards/** - Standards for installing and managing software on desktops
- **user-profiles-and-permissions/** - User profile and permission management on desktops
- **updates-and-patching/** - Update and patch management procedures for desktops

## Scope

Covers desktop workstation management to ensure consistent, secure, and productive work environments.
