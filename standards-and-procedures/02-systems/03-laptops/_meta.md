# Laptop Standards

## Domain

This directory contains standards specific to laptop computers, including encryption, remote access, offline data handling, and decommissioning.

## Contents

- **encryption-and-locking-policy/** - Encryption requirements and screen locking policies for laptops
- **vpn-requirements/** - VPN configuration and usage requirements for laptops
- **offline-data-handling/** - Standards for handling data when working offline
- **portable-device-decommissioning/** - Procedures for securely decommissioning laptops

## Scope

Covers laptop-specific security and operational requirements to address the unique risks of portable devices.
