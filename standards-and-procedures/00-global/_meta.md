# Global Standards

## Domain

This directory contains global standards that apply across all areas of CloudCurio operations. These foundational documents establish baseline requirements, terminology, and procedures that should be followed organization-wide.

## Contents

- **glossary/** - Standard terminology and definitions used throughout CloudCurio
- **document-style-guide/** - Guidelines for writing and formatting documentation
- **classification-and-labeling/** - Standards for classifying and labeling data, systems, and resources
- **roles-and-responsibilities/** - Definition of organizational roles and their responsibilities
- **standard-change-control/** - Standard procedures for managing changes across the organization
- **risk-and-impact-matrix/** - Framework for assessing risks and impacts of decisions and changes

## Scope

These standards provide the foundation for all other standards and procedures in the organization. They should be consulted first when creating new documentation or implementing new processes.
