# Software Standards

## Domain

This directory contains standards for software applications, programming practices, and database management.

## Contents

- **01-applications/** - Standards for application software management and deployment
- **02-programming/** - Software development standards including coding, testing, and documentation
- **03-databases/** - Database management standards, design guidelines, and best practices

## Scope

Covers all aspects of software development, deployment, and management to ensure quality, security, and maintainability.
