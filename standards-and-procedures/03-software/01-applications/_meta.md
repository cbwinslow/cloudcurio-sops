# Application Standards

## Domain

This directory contains standards for managing application software, including approval, installation, versioning, and sandboxing.

## Contents

- **approved-software-list/** - List of approved applications and approval process
- **installation-guidelines/** - Guidelines for installing and configuring applications
- **versioning-and-deprecation-policy/** - Policies for managing application versions and deprecating old software
- **application-sandboxing/** - Standards for isolating and sandboxing applications

## Scope

Ensures consistent and secure management of application software across the organization.
